Welcome to the SHRDLURN+ game

Thank you for your participation!

  • You should interact with the computer using natural language phrases.

  • You can use any type of language, but try to be consistent (e.g., do not use synonyms in different iterations of the game).

  • In each iteration of the game, you will have a start and a goal state, where you need to give instructions to the system and then choose one of the following presented states.

  • The system performs one action at a time, so for some iterations you will have to give multiple (separated) instructions.
